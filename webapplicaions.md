## Тема: Формы HTML

Форма - это какой-то элемент, который служит для ввода информации с целью её дальнейшей обработки.


@media
- speech        - Для слепых
- handler       - Сенсорные устройства
- print         - При печати
- projection    - При выводе на проектор
- screen        - Для мониторов


```css

@media handler {
    /* For tablets and phones */
}

@media screen {
    /* For desktop */
}

```
